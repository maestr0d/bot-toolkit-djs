0.0.11 Changelog:

  • Added the ability to message users (You can only save presets for server messages for now. Next version will have support for DM preset saving and viewing DM replies)

  • Added new modular collapsible system
  • Moved footer credits to the About tab
  • Moved changelog side panel to it's own prompt for easier reading
  • TAB key now inserts an indent in the text area
  • Prompt values now reset when prompts are closed
  • Logging improvements
  • Code adjustments/optimizations
  • Quite a few CSS/HTML adjustments
